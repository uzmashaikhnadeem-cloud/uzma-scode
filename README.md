# uzma-scode

## Project 1: Expense Tracker 💸

A simple Python-based expense tracking application that helps you manage your shopping expenses efficiently.

### Features
- ✅ Add expenses with date, category, description, and amount
- ✅ View all recorded expenses
- ✅ Calculate total spending
- ✅ User-friendly menu interface

### How to Run

#### Prerequisites
- Python 3.x installed on your system

#### Steps to Run the Code

1. **Clone the repository**
   ```bash
   git clone https://github.com/uzmashaikhnadeem-cloud/uzma-scode.git
   cd uzma-scode
   ```

2. **Run the expense tracker**
   ```bash
   python "project 1 expense tracker.py"
   ```
   
   Or if using Python 3:
   ```bash
   python3 "project 1 expense tracker.py"
   ```

3. **Follow the on-screen menu**
   - Enter `1` to add new expenses
   - Enter `2` to view all expenses you've recorded
   - Enter `3` to see your total spending
   - Enter `4` to exit the application
